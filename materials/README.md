# 材料选择

> 不同项目适合不同材料，以下是我们常用的材料类型及选择建议。

---

## 快速选择指南

| 项目类型 | 推荐材料 | 理由 |
|----------|----------|------|
| 室内海报 | 相纸 / PP | 成本低，色彩好 |
| 户外横幅 | 防水布 | 耐候性强 |
| 立体字 | 亚克力 / 金属 | 质感高级，耐用 |
| 灯箱面 | 亚克力 + LED | 透光均匀 |
| 窗贴 | 透明乙烯基 / 磨砂膜 | 不影响采光 |
| 车身 | 高级铸塑膜 | 拉伸性好，贴合曲面 |

---

## 材料类别

- [**乙烯基材料**](vinyl) — 最常用，贴纸、窗贴、车身
- [**亚克力**](acrylic) — 灯箱面板、立体字、标牌
- [**金属**](metal) — 户外字、高档标识、门牌
