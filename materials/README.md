# 材料参考

> 常用材料规格及用途

---

## PVC 板（Foam PVC）

| 厚度 | 用途 |
|------|------|
| **1/8"** | 活动板、A字板（A-Frame）图形或替换面，可贴 Vinyl |
| **1/2"** | 太厚只能 CNC 切割，多数用于广告招牌制作 |

---

## 亚克力板（Acrylic / Plexiglass）

| 厚度 | 用途 |
|------|------|
| **1/8"** | 薄板，适用小标牌 |
| **1/4"** | 灯箱制作（常用），白色/黑色/透明 |
| **1/2"** | 厚重标牌 |

- 切割方式：**激光镭射切割**
- 常用颜色：白色、黑色、透明

---

## 铝片（Aluminum）

| 厚度 | 用途 |
|------|------|
| **0.040"** | 常用厚度，贴 Vinyl 制作各种牌子 |
| **0.060"** | 更厚更坚固，适合大型/户外标牌 |

## 泡沫板（Foam Board）

| 厚度 | 用途 |
|------|------|
| **3/16"** | 轻质展示板，适合室内标牌、摄影背景、临时展示 |
| **1/2"** | 加厚版，更坚固，适合需要一定支撑力的室内展示 |

- 材质：聚苯乙烯泡沫芯 + 纸面/塑料面
- 轻便易裁切，适合短期/室内使用
- ⚠️ 不如 PVC 板耐用防水，户外需覆膜或保护
